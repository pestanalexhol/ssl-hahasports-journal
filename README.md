# ssl-hahasports-journal

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定网站或域名，仅作为静态页面集合的存储与展示用途。

## 项目简介

该项目旨在整理和保存一系列独立的 HTML 文件，这些文件可能包含信息展示、工具页面、存档记录等内容。每个页面均为自包含的 HTML 文件，不依赖外部数据库或后端服务。

## 目录结构

```
ssl-hahasports-journal/
├── pages/          # 存放主要 HTML 页面文件
├── assets/         # 页面引用的静态资源（图片、样式、脚本等）
├── archive/        # 历史版本或归档页面
└── README.md       # 本文件
```

- `pages/`：当前活跃的 HTML 页面，每个文件为一个独立页面。
- `assets/`：页面所需的公共资源文件，按类型组织。
- `archive/`：不再更新但保留作为参考的旧版本页面。

## 页面归档说明

- 所有页面均为纯前端 HTML 文件，可直接在浏览器中打开。
- 页面内容由作者定期整理和更新，部分页面可能标记为“归档”状态。
- 归档页面保留原始内容，不再进行修改。
- 每个页面文件命名应尽量反映其内容或用途。
